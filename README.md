# D.R.P. Convent Public School Website

A modern, responsive school website built with **React (Vite)** and **Node.js (Express)**.

## 🚀 Deployment Status
- **Backend**: Node.js + Express + MongoDB
- **Frontend**: React + Vite + Tailwind/CSS
- **Deployed On**: Vercel (Recommended)

## ✨ Features
- **3D UI**: Glassmorphism cards and animations.
- **Admin Portal**: Secure management for results and notices.
- **Dynamic Content**: Live Notice Board and Gallery.

## 🛠️ Setup
1. `npm install`
2. `npm run dev:full`
